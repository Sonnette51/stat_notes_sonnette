    模块 5：生成函数与特征函数¶

    对应 Lec 11：概率母函数（PGF）、矩母函数（MGF）、特征函数（CF）的定义、性质与应用；随机和定理。含于"Lec 11–14 约 35 分"份额内。

    Part A：公式与方法速查¶

    A.1 概率母函数 PGF¶

    定义：设 \(X\) 取非负整数值，其概率母函数为

    \[ g(s) = E(s^X) = \sum_{j=0}^{\infty} s^j \mathbb{P}(X=j), \quad s \in [-1,1] \]

    约定 \(0^0=1\)。级数在 \([-1,1]\) 上绝对收敛，且 \(g(1)=1\)。

    核心性质（设 \(g^{(k)}\) 为 \(k\) 阶导数）：

    性质 公式 用途
    恢复分布 \(\mathbb{P}(X=k) = \dfrac{g^{(k)}(0)}{k!}\) 由 PGF 反推分布列（等价于读 Taylor 展开系数）
    期望 \(E(X) = g'(1)\) 求矩
    方差 \(\operatorname{var}(X) = g''(1) + g'(1) - [g'(1)]^2\) \(g''(1)=E[X(X-1)]\) 是阶乘矩
    线性变换 \(g_{aX+b}(s) = s^b\, g_X(s^a)\) \(aX+b\) 仍取非负整值时适用
    独立和 \(g_{X_1+\cdots+X_n}(s) = g_1(s)\cdots g_n(s)\) 求独立和的分布（再配合可逆性识别）
    可逆性 PGF 与分布列相互唯一决定 比对 PGF 形状即可断言分布

    随机和定理（定理 1.1，高频考点）：\(\{X_j\}\) 独立同分布、PGF 为 \(g_X\)；\(N\) 取正整数值、PGF 为 \(g_N\)，且 \(N\) 与 \(\{X_j\}\) 独立。则 \(W = X_1 + \cdots + X_N\) 的 PGF 为复合

    \[ g_W(s) = g_N\bigl[g_X(s)\bigr] \]

    证明依据是全期望公式：\(E(s^W) = E\bigl[E(s^W \mid N)\bigr] = \sum_{n\ge 1} [g_X(s)]^n \mathbb{P}(N=n)\)。

    二维 PGF：\(g(s,t) = E(s^X t^Y)\)，要点：

    • \(X,Y\) 独立 \(\iff g(s,t) = g_X(s)\,g_Y(t)\) 对所有 \(s,t\) 成立；
    • 边缘：\(g(s,1)=g_X(s)\)，\(g(1,t)=g_Y(t)\)；
    • \(E(XY) = \left.\dfrac{\partial^2 g}{\partial s\,\partial t}\right|_{s=t=1}\)，从而 \(\operatorname{cov}(X,Y) = g_{st}(1,1) - g_s(1,1)\,g_t(1,1)\)。

    [!WARNING] 易错 PGF 只对非负整值随机变量有定义；连续型（如指数、正态）没有 PGF。求矩时 PGF 在 \(s=1\) 处求导，MGF 在 \(s=0\) 处求导，二者不要混淆。另外 \(g''(1)\) 给出的是阶乘矩 \(E[X(X-1)]\) 而不是 \(E(X^2)\)，所以方差公式里多出一项 \(g'(1)\)。

    A.2 矩母函数 MGF¶

    定义：

    \[ M_X(s) = E(e^{sX}) = \begin{cases} \sum_j e^{s x_j}\,\mathbb{P}(X=x_j) & \text{离散型} \\ \displaystyle\int_{-\infty}^{+\infty} e^{sx} f_X(x)\,dx & \text{连续型} \end{cases} \]

    仅当 \(E(e^{sX}) < \infty\) 时存在。Taylor 展开给出"矩的母函数"：

    \[ M(s) = \sum_{k=0}^{\infty} \frac{s^k}{k!} E(X^k) \]

    核心性质：

    性质 公式 用途
    归一化 \(M(0)=1\) 检验答案、定常数
    提取矩 \(E(X^k) = M^{(k)}(0)\) 求导代 \(0\)，或直接读展开式 \(s^k\) 系数乘 \(k!\)
    方差 \(\operatorname{var}(X) = M''(0) - [M'(0)]^2\)
    线性变换 \(M_{aX+b}(s) = e^{sb} M_X(as)\) 标准化、平移缩放
    独立和 \(M_{X_1+\cdots+X_n}(s) = M_1(s)\cdots M_n(s)\) 求独立和的分布
    可逆性 若 \(M(s)\) 在 \(0\) 的某邻域 \([-a,a]\) 内有限，则唯一决定分布 比对 MGF 识别分布
    取 \(0\) 概率 \(X\) 取非负整值时 \(\mathbb{P}(X=0) = \lim_{s\to-\infty} M(s)\) 配合条件期望 \(E(X \mid X \ne 0)\) 类问题

    随机和（MGF 版）：条件同定理 1.1，则

    \[ M_W(s) = \sum_{n=1}^{\infty} [M_X(s)]^n\, \mathbb{P}(N=n) = g_N\bigl[M_X(s)\bigr] \]

    即把 \(N\) 的 PGF 中的变量替换为 \(M_X(s)\)（等价于在 \(M_N(s)\) 中把 \(e^s\) 替换为 \(M_X(s)\)）。

    多元 MGF（与多元正态衔接）：\(M_{\vec X}(\vec s) = E(e^{\vec s'\vec X})\)，要点：边缘 MGF 由其余分量取 \(0\) 得到；\(X_1,\ldots,X_n\) 独立 \(\iff M_{\vec X}(\vec s) = \prod_i M_{X_i}(s_i)\)；线性变换 \(\vec Y = A\vec X + \vec b\) 时 \(M_{\vec Y}(\vec t) = e^{\vec t'\vec b} M_{\vec X}(A'\vec t)\)；多元正态 \(\mathcal{N}(\vec\mu,\Sigma)\) 的 MGF 为 \(\exp\bigl(\vec s'\vec\mu + \tfrac12 \vec s'\Sigma\vec s\bigr)\)。

    [!WARNING] 易错 MGF 不一定存在：指数分布的 MGF 仅在 \(s<\lambda\) 时有限；几何分布要求 \(s<-\ln q\)；Cauchy 分布的 MGF 对任意 \(s\ne 0\) 都不存在（重尾使积分发散）。引用可逆性时必须有"在 \(0\) 的邻域内有限"这一前提。随机和公式是 \(g_N[M_X(s)]\)（外层是 \(N\) 的 PGF），不要写成 \(M_N[M_X(s)]\)。

    A.3 特征函数 CF¶

    定义：对任意随机变量 \(X\)，

    \[ \phi(t) = E(e^{itX}) = E\cos(tX) + i\,E\sin(tX), \quad t \in \mathbb{R} \]

    连续型时 \(\phi(t) = \int_{-\infty}^{+\infty} e^{itx} f(x)\,dx\)，即密度的 Fourier 变换。因 \(|e^{itX}|=1\)，特征函数对任何随机变量都存在。

    基本性质：

    # 性质 说明
    1 \(\lvert\phi(t)\rvert \le \phi(0) = 1\)，\(\phi(-t) = \overline{\phi(t)}\) 有界、共轭对称
    2 \(\phi(t)\) 在 \(\mathbb{R}\) 上一致连续
    3 若 \(E\lvert X\rvert^k < \infty\)，则 \(\phi^{(k)}(0) = i^k E(X^k)\)，即 \(E(X^k) = \phi^{(k)}(0)/i^k\) 与矩的关系
    4 \(\phi_{aX+b}(t) = e^{itb}\,\phi_X(at)\) 线性变换
    5 \(X_1,\ldots,X_n\) 独立 \(\Rightarrow \phi_{X_1+\cdots+X_n}(t) = \prod_k \phi_k(t)\) 独立和

    与分布的关系：特征函数与分布函数相互唯一决定（逆转公式属选修，不考其细节，但"唯一决定"这一结论可直接使用）。

    矩展开：若 \(E(X^2)<\infty\)，则

    \[ \phi(t) = 1 + itE(X) - \frac{1}{2}t^2 E(X^2) + o(t^2) \]

    混合分布（定理 3.4）：若 \(Y\) 以概率 \(\lambda_k\) 服从分布 \(F_k\)（\(\lambda_k \ge 0\)，\(\sum_k \lambda_k = 1\)），则 \(\phi_Y(t) = \sum_k \lambda_k \phi_k(t)\)。同样的凸组合结构对 MGF、PGF 也成立。

    连续性定理（定理 3.6）：

    \[ X_n \xrightarrow{d} X \iff \lim_{n\to\infty} \phi_n(t) = \phi(t), \quad \forall t \in \mathbb{R} \]

    这是模块 6 中证明 CLT 的关键工具。

    [!WARNING] 易错 三个高频判断题陷阱： 1. "MGF 总存在"——错，Cauchy 分布是反例；而 CF 恒存在。 2. "\(\phi_{X+Y}(t) = \phi_X(t)\phi_Y(t) \Rightarrow X,Y\) 独立"——错。反例：\(X\) 服从 Cauchy 分布，\(Y = aX\)（\(a>0\)），则 \(\phi_{X+Y}(t) = e^{-(1+a)\lvert t\rvert} = \phi_X(t)\phi_Y(t)\)，但 \(Y\) 是 \(X\) 的函数，显然不独立。独立的充要条件是联合 CF 分解：\(\phi(t_1,t_2) = \phi_X(t_1)\phi_Y(t_2)\)。 3. "混合分布 = 线性组合"——错。\(\sum_k \lambda_k F_k\) 表示以概率 \(\lambda_k\) 服从 \(F_k\)，与随机变量的线性组合 \(\sum_k \lambda_k X_k\) 是完全不同的对象。

    A.4 常见分布的 PGF / MGF / CF 总表¶

    记 \(q = 1-p\)。卷首会提供重要分布的 MGF；熟悉分布的三种母函数可直接引用，不熟悉的分布需会从定义计算。

    分布 PGF \(g(s)=E(s^X)\) MGF \(M(s)=E(e^{sX})\)（存在范围） CF \(\phi(t)=E(e^{itX})\)
    退化 \(X=c\) \(s^c\)（\(c\in\mathbb{N}\)） \(e^{cs}\)（\(\forall s\)） \(e^{ict}\)
    Bernoulli \(B(1,p)\) \(q+ps\) \(q+pe^s\)（\(\forall s\)） \(q+pe^{it}\)
    二项 \(B(n,p)\) \((q+ps)^n\) \((q+pe^s)^n\)（\(\forall s\)） \((q+pe^{it})^n\)
    Poisson \(\mathcal{P}(\lambda)\) \(e^{\lambda(s-1)}\) \(e^{\lambda(e^s-1)}\)（\(\forall s\)） \(e^{\lambda(e^{it}-1)}\)
    几何 \(G(p)\)（从 \(1\) 起） \(\dfrac{ps}{1-qs}\) \(\dfrac{pe^s}{1-qe^s}\)（\(s<-\ln q\)） \(\dfrac{pe^{it}}{1-qe^{it}}\)
    Pascal（负二项，\(m\) 阶） \(\left(\dfrac{ps}{1-qs}\right)^m\) \(\left(\dfrac{pe^s}{1-qe^s}\right)^m\)（\(s<-\ln q\)） \(\left(\dfrac{pe^{it}}{1-qe^{it}}\right)^m\)
    均匀 \(U(a,b)\) — \(\dfrac{e^{sb}-e^{sa}}{s(b-a)}\)（\(s\ne 0\)，\(M(0)=1\)） \(\dfrac{e^{itb}-e^{ita}}{it(b-a)}\)
    指数 \(\mathcal{E}(\lambda)\) — \(\dfrac{\lambda}{\lambda-s}\)（\(s<\lambda\)） \(\left(1-\dfrac{it}{\lambda}\right)^{-1}\)
    Gamma \(\Gamma(\alpha,\lambda)\) — \(\left(\dfrac{\lambda}{\lambda-s}\right)^{\alpha}\)（\(s<\lambda\)） \(\left(1-\dfrac{it}{\lambda}\right)^{-\alpha}\)
    正态 \(\mathcal{N}(\mu,\sigma^2)\) — \(e^{\mu s + \sigma^2 s^2/2}\)（\(\forall s\)） \(e^{i\mu t - \sigma^2 t^2/2}\)
    Cauchy — 不存在（仅 \(s=0\)） \(e^{-\lvert t\rvert}\)

    记忆口诀：三者只差一个替换——在 PGF 中令 \(s \to e^s\) 得 MGF，令 \(s \to e^{it}\) 得 CF（对非负整值分布成立）；在 MGF 中令 \(s \to it\) 得 CF。

    A.5 三种工具对比¶

    PGF（非负整值） MGF（需存在） CF（恒存在）
    定义 \(g(s)=E(s^X)\) \(M(s)=E(e^{sX})\) \(\phi(t)=E(e^{itX})\)
    归一点 \(g(1)=1\) \(M(0)=1\) \(\phi(0)=1\)
    提取矩 \(g'(1)=EX\)，\(g''(1)=E[X(X-1)]\) \(M^{(k)}(0)=E(X^k)\) \(\phi^{(k)}(0)=i^k E(X^k)\)
    恢复分布 \(\mathbb{P}(X=k)=g^{(k)}(0)/k!\) 可逆（邻域内有限时） 与 CDF 一一对应
    独立和 \(\prod g_k(s)\) \(\prod M_k(s)\) \(\prod \phi_k(t)\)
    线性变换 \(aX+b\) \(s^b g(s^a)\) \(e^{sb}M(as)\) \(e^{itb}\phi(at)\)
    随机和 \(S_N\) \(g_N[g_X(s)]\) \(g_N[M_X(s)]\) \(g_N[\phi_X(t)]\)

    A.6 由生成函数识别分布的套路¶

    方法：把所得母函数整理成标准形状，与总表逐一比对，再引用可逆性下结论。常见形状：

    看到的形状 识别为 备注
    \((q+ps)^n\) 或 \((q+pe^s)^n\) \(B(n,p)\) 括号内一次式
    \(e^{c(s-1)}\) 或 \(e^{c(e^s-1)}\) \(\mathcal{P}(c)\) 指数上"括号减一"
    \(\dfrac{as}{1-(1-a)s}\) \(G(a)\) 整理成 \(\dfrac{ps}{1-qs}\) 后读出 \(p\)
    \(\dfrac{c}{c-s}\) \(\mathcal{E}(c)\)
    \(\left(\dfrac{c}{c-s}\right)^{\alpha}\) \(\Gamma(\alpha,c)\) 指数分布的 \(\alpha\) 次幂
    \(e^{as+bs^2}\)（\(b>0\)） \(\mathcal{N}(a,\,2b)\) 对照 \(e^{\mu s+\sigma^2 s^2/2}\)
    \(\sum_k p_k M_k(s)\)（凸组合） 混合分布 以概率 \(p_k\) 服从第 \(k\) 个分布
    多项式 \(\sum_k p_k e^{ks}\) 离散分布列 \(e^{ks}\) 的系数就是 \(\mathbb{P}(X=k)\)

    操作流程（题型四：随机和）：① 写出 \(g_X\)（或 \(M_X\)）；② 写出 \(g_N\)；③ 复合 \(g_W = g_N[g_X(s)]\)；④ 化简并比对总表识别分布。

    操作流程（题型五：由 MGF 反推分布）：若 MGF 是若干已知 MGF 的凸组合，则为混合分布，权重即选中概率；若 \(X\) 取非负整值，把 \(M(s)\) 展开成 \(\sum_k p_k e^{ks}\)，读系数得分布列。

    Part B：判断题精选¶

    B1 任何随机变量都有矩母函数，也都有特征函数。

    [!TIP]- 答案 错。特征函数对任何随机变量恒存在（因 \(|e^{itX}|=1\)），但 MGF 要求 \(E(e^{sX})<\infty\)。反例：Cauchy 分布（密度 \(f(x)=\frac{1}{\pi(1+x^2)}\)）对任意 \(s\ne 0\) 都有 \(E(e^{sX})=\infty\)，MGF 不存在，但其 CF 为 \(e^{-\lvert t\rvert}\)。

    B2 若 \(\phi_{X+Y}(t) = \phi_X(t)\,\phi_Y(t)\) 对所有 \(t\) 成立，则 \(X\) 与 \(Y\) 相互独立。

    [!TIP]- 答案 错。反例：\(X\) 服从 Cauchy 分布，\(Y = aX\)（\(a>0\)）。则 \(\phi_X(t)=e^{-\lvert t\rvert}\)，\(\phi_Y(t)=e^{-a\lvert t\rvert}\)，而 \(X+Y=(1+a)X\) 的 CF 为 \(e^{-(1+a)\lvert t\rvert} = \phi_X(t)\phi_Y(t)\)，但 \(Y\) 是 \(X\) 的函数，显然不独立。独立的充要条件是联合特征函数分解 \(\phi(t_1,t_2)=\phi_X(t_1)\phi_Y(t_2)\)，仅和的 CF 分解不够。

    B3 设 \(X,Y\) 的 MGF 在 \(0\) 的某邻域内有限且处处相等，则 \(X\) 与 \(Y\) 同分布。

    [!TIP]- 答案 对。这正是 MGF 的可逆性（唯一性）：若存在 \(a>0\) 使 \(M(s)<\infty\) 对所有 \(s\in[-a,a]\) 成立，则 \(M\) 唯一决定分布。注意"邻域内有限"这一前提不可省略。

    B4 对任意随机变量 \(X\) 都可以定义概率母函数 \(g(s)=E(s^X)\)，且 \(E(X)=g'(0)\)。

    [!TIP]- 答案 错，两处都错。其一，PGF 只对取非负整数值的随机变量定义（否则 \(s^X\) 可能无意义或级数结构丢失），连续型分布没有 PGF；其二，即使 PGF 存在，求期望应在 \(s=1\) 处求导：\(E(X)=g'(1)\)，而 \(g^{(k)}(0)/k!\) 给出的是 \(\mathbb{P}(X=k)\)。

    B5 设 \(X\) 取非负整数值，PGF 为 \(g\)，MGF 为 \(M\)，则 \(\mathbb{P}(X=0) = g(0) = \lim_{s\to-\infty} M(s)\)。

    [!TIP]- 答案 对。\(g(0) = \sum_{j\ge 0} 0^j\,\mathbb{P}(X=j) = \mathbb{P}(X=0)\)（约定 \(0^0=1\)）；而 \(M(s) = \sum_{j\ge 0} e^{sj}\,\mathbb{P}(X=j)\) 中，当 \(s\to-\infty\) 时 \(j\ge 1\) 的各项 \(e^{sj}\to 0\)（由控制收敛可逐项取极限），仅剩 \(j=0\) 项，故极限为 \(\mathbb{P}(X=0)\)。这一性质在求 \(E(X \mid X\ne 0)\) 时很有用。

    题 5.1（来源：作业 HW12 第 1 题）¶

    题目：设 \(X \sim U(0,1)\) 为均匀分布，从定义计算 \(X\) 的特征函数。

    [!EXAMPLE]- 详细解答 方法点拨：不熟悉的分布从定义 \(\phi(t)=E(e^{itX})\) 直接积分。

    解：\(X\) 的密度为 \(f(x)=1\)，\(x\in(0,1)\)。由特征函数定义，

    \[ \phi(t) = E(e^{itX}) = \int_0^1 e^{itx}\,dx \]

    当 \(t \ne 0\) 时，对 \(e^{itx}\) 求原函数 \(\dfrac{e^{itx}}{it}\)，由 Newton–Leibniz 公式，

    \[ \phi(t) = \left.\frac{e^{itx}}{it}\right|_{0}^{1} = \frac{e^{it}-1}{it} \]

    当 \(t = 0\) 时，\(\phi(0) = E(e^{0}) = 1\)（也与上式当 \(t\to 0\) 的极限一致，因为 \(e^{it}-1 \sim it\)）。

    答：\(\phi(t) = \dfrac{e^{it}-1}{it}\)（\(t\ne 0\)），\(\phi(0)=1\)。

    [!TIP] 技巧/易错 这与总表中 \(U(a,b)\) 的 CF \(\dfrac{e^{itb}-e^{ita}}{it(b-a)}\) 在 \(a=0,b=1\) 时一致，可用来检验。写答案时不要漏掉 \(t=0\) 的情形；并且任何 CF 都应满足 \(\phi(0)=1\)、\(\lvert\phi(t)\rvert\le 1\)，可作为快速自查。

    题 5.2（来源：作业 HW11 第 3 题，选做）¶

    题目：设 \(p=1-q\in(0,1)\)，\(X\) 的分布列为

    \[ \mathbb{P}(X=k) = -\frac{q^k}{k\ln p}, \quad k=1,2,\ldots \]

    （对数分布）。证明其 PGF 为 \(g_X(s) = \dfrac{\ln(1-qs)}{\ln p}\)，并求 \(E(X) = -\dfrac{q}{p\ln p}\)。

    [!EXAMPLE]- 详细解答 方法点拨：从定义逐项求和，认出 \(-\ln(1-x)=\sum_{k\ge1} x^k/k\) 这一标准级数。

    解：第一步，按 PGF 定义（\(|s|\le 1\) 时级数绝对收敛）：

    \[ g_X(s) = \sum_{k=1}^{\infty} s^k\,\mathbb{P}(X=k) = \sum_{k=1}^{\infty} s^k \left(-\frac{q^k}{k\ln p}\right) = -\frac{1}{\ln p}\sum_{k=1}^{\infty} \frac{(qs)^k}{k} \]

    第二步，由对数级数 \(\sum_{k=1}^{\infty} \dfrac{x^k}{k} = -\ln(1-x)\)（\(|x|<1\)），取 \(x = qs\)：

    \[ g_X(s) = -\frac{1}{\ln p}\bigl[-\ln(1-qs)\bigr] = \frac{\ln(1-qs)}{\ln p} \]

    验证：\(g_X(1) = \dfrac{\ln(1-q)}{\ln p} = \dfrac{\ln p}{\ln p} = 1\)，符合 PGF 归一性，说明这确实是一个概率分布。

    第三步，由 PGF 求期望公式 \(E(X) = g'(1)\)。先求导（链式法则）：

    \[ g_X'(s) = \frac{1}{\ln p}\cdot\frac{-q}{1-qs} \]

    代入 \(s=1\)，并用 \(1-q=p\)：

    \[ E(X) = g_X'(1) = \frac{1}{\ln p}\cdot\frac{-q}{1-q} = -\frac{q}{p\ln p} \]

    注意 \(\ln p<0\)，故 \(E(X)>0\)，符号合理。

    答：\(g_X(s) = \dfrac{\ln(1-qs)}{\ln p}\)，\(E(X) = -\dfrac{q}{p\ln p}\)。

    [!TIP] 技巧/易错 "不熟悉分布从定义算"的典型题：核心是把 \(\sum s^k p_k\) 凑成已知幂级数（几何级数、指数级数、对数级数、广义二项级数）。算完一定代 \(s=1\) 检验是否等于 \(1\)。

    题 5.3（来源：样卷 题五(4)）¶

    题目：设 \(Z \sim \mathcal{N}(0,1)\)。请利用 \(Z\) 的矩母函数求 \(E(Z^4)\) 与 \(E(Z^6)\)。

    [!EXAMPLE]- 详细解答 方法点拨：把 \(M(s)=e^{s^2/2}\) 展成幂级数，用 \(E(X^k) = k!\times(s^k\text{ 的系数})\) 读矩，比连续求导六次省力得多。

    解：标准正态的 MGF 为 \(M(s) = e^{s^2/2}\)（卷首给出，可直接引用）。

    第一步，对 \(e^{x}\) 用指数级数 \(e^x = \sum_{k\ge 0} x^k/k!\)，取 \(x = s^2/2\)：

    \[ M(s) = \sum_{k=0}^{\infty} \frac{(s^2/2)^k}{k!} = \sum_{k=0}^{\infty} \frac{s^{2k}}{2^k\,k!} \]

    第二步，由 MGF 的矩展开 \(M(s) = \sum_{n\ge 0} \dfrac{E(Z^n)}{n!}s^n\)，比较两个展开式中 \(s^n\) 的系数：

    \[ \frac{E(Z^n)}{n!} = \begin{cases} \dfrac{1}{2^{n/2}(n/2)!} & n \text{ 为偶数} \\ 0 & n \text{ 为奇数} \end{cases} \]

    第三步，代入 \(n=4\)（即 \(k=2\)）：

    \[ E(Z^4) = \frac{4!}{2^2\cdot 2!} = \frac{24}{8} = 3 \]

    代入 \(n=6\)（即 \(k=3\)）：

    \[ E(Z^6) = \frac{6!}{2^3\cdot 3!} = \frac{720}{48} = 15 \]

    答：\(E(Z^4)=3\)，\(E(Z^6)=15\)。

    [!TIP] 技巧/易错 一般地 \(E(Z^{2k}) = (2k-1)!! = 1\cdot3\cdot5\cdots(2k-1)\)，奇数阶矩为 \(0\)。考试中若题目明确要求"利用 MGF"，必须写出级数比较或求导过程，不能只背结论。

    题 5.4（来源：作业 HW11 第 7 题）¶

    题目：设 \(X,Y\) 独立同分布，矩母函数为 \(M(s) = e^{s+s^2}\)，\(s\in\mathbb{R}\)。令 \(Z = X - Y\)。求：(1) \(Z\) 服从什么分布；(2) \(E(Z^4)\)。

    [!EXAMPLE]- 详细解答 方法点拨：先比对 \(e^{\mu s+\sigma^2 s^2/2}\) 识别 \(X\)，再用线性变换+独立和求 \(M_Z\)，最后用可逆性识别 \(Z\)。

    解：(1) 第一步，识别 \(X\) 的分布。把 \(M(s) = e^{s+s^2}\) 与正态 MGF \(e^{\mu s + \sigma^2 s^2/2}\) 比对：\(\mu = 1\)，\(\sigma^2/2 = 1\) 即 \(\sigma^2 = 2\)。\(M(s)\) 在 \(0\) 的邻域内有限，由 MGF 可逆性，\(X,Y \sim \mathcal{N}(1,2)\)。

    第二步，求 \(-Y\) 的 MGF。由线性变换性质 \(M_{aX+b}(s) = e^{sb}M(as)\)，取 \(a=-1,b=0\)：

    \[ M_{-Y}(s) = M(-s) = e^{-s+s^2} \]

    第三步，\(X\) 与 \(-Y\) 独立（\(X,Y\) 独立的推论），由独立和性质：

    \[ M_Z(s) = M_X(s)\,M_{-Y}(s) = e^{s+s^2}\cdot e^{-s+s^2} = e^{2s^2} \]

    第四步，比对正态 MGF：\(\mu = 0\)，\(\sigma^2/2 = 2\) 即 \(\sigma^2 = 4\)。由可逆性，\(Z \sim \mathcal{N}(0,4)\)。

    (2) 法一（标准化）：\(Z = 2V\)，\(V\sim\mathcal{N}(0,1)\)，故由题 5.3 的结论 \(E(V^4)=3\)：

    \[ E(Z^4) = 2^4\,E(V^4) = 16\times 3 = 48 \]

    法二（直接展开 \(M_Z\)）：\(M_Z(s) = e^{2s^2} = \sum_{k\ge 0} \dfrac{2^k s^{2k}}{k!}\)，\(s^4\) 的系数为 \(\dfrac{2^2}{2!} = 2\)，故 \(E(Z^4) = 4!\times 2 = 48\)。两法一致。

    答：(1) \(Z \sim \mathcal{N}(0,4)\)；(2) \(E(Z^4) = 48\)。

    [!TIP] 技巧/易错 \(X-Y\) 的 MGF 是 \(M_X(s)M_Y(-s)\)，不是 \(M_X(s)/M_Y(s)\)，也不能写成 \(M_X(s)M_Y(s)\)。独立正态相减，均值相减、方差相加：\(\operatorname{var}(Z)=2+2=4\)，这是高频易错点。

    题 5.5（来源：作业 HW11 第 4 题，教材 4.33）¶

    题目：连续型随机变量 \(X\) 的矩母函数为

    \[ M(s) = \frac{1}{3}\cdot\frac{2}{2-s} + \frac{2}{3}\cdot\frac{3}{3-s} \]

    求 \(X\) 的概率密度函数。

    [!EXAMPLE]- 详细解答 方法点拨：MGF 是已知 MGF 的凸组合 \(\Rightarrow\) 混合分布，权重即选中概率。

    解：第一步，认出结构。\(\dfrac{2}{2-s}\) 是 \(\mathcal{E}(2)\) 的 MGF（\(s<2\)），\(\dfrac{3}{3-s}\) 是 \(\mathcal{E}(3)\) 的 MGF（\(s<3\)），系数 \(\dfrac13 + \dfrac23 = 1\) 且均非负，故 \(M(s)\) 是两个指数分布 MGF 的凸组合。

    第二步，由混合分布定理（凸组合的母函数对应混合分布）及 MGF 可逆性（\(M\) 在 \(0\) 的邻域 \(s<2\) 内有限），\(X\) 是以概率 \(\dfrac13\) 服从 \(\mathcal{E}(2)\)、以概率 \(\dfrac23\) 服从 \(\mathcal{E}(3)\) 的混合变量，密度为对应密度的同权重组合：

    \[ f_X(x) = \frac{1}{3}\cdot 2e^{-2x} + \frac{2}{3}\cdot 3e^{-3x} = \left(\frac{2}{3}e^{-2x} + 2e^{-3x}\right), \quad x \ge 0 \]

    \(x<0\) 时 \(f_X(x) = 0\)。

    检验：\(\int_0^\infty f_X = \dfrac13 + \dfrac23 = 1\)；\(M(0) = \dfrac13+\dfrac23 = 1\)。

    答：\(f_X(x) = \dfrac{2}{3}e^{-2x} + 2e^{-3x}\)（\(x\ge 0\)），否则为 \(0\)。

    [!TIP] 技巧/易错 混合分布是"密度的凸组合"，不是"随机变量的线性组合"：\(X\) 不等于 \(\frac13 X_1 + \frac23 X_2\)。识别关键是权重非负且和为 \(1\)。

    题 5.6（来源：作业 HW11 第 5 题，教材 4.35）¶

    题目：设 \(X\) 取非负整数值，其矩母函数为

    \[ M_X(s) = c\cdot\frac{3 + 4e^{2s} + 2e^{3s}}{3 - e^s} \]

    其中 \(c\) 为常数。求 \(E(X)\)、\(p_X(1)\)，以及 \(E(X \mid X \ne 0)\)。

    [!EXAMPLE]- 详细解答 方法点拨：先用 \(M(0)=1\) 定 \(c\)；求概率用几何级数把 \(M\) 展成 \(\sum p_k e^{ks}\) 读系数；\(E(X\mid X\ne 0)\) 用 \(\mathbb{P}(X=0)=\lim_{s\to-\infty}M(s)\)。

    解：第一步，定常数 \(c\)。由归一化 \(M_X(0)=1\)：

    \[ M_X(0) = c\cdot\frac{3+4+2}{3-1} = \frac{9c}{2} = 1 \implies c = \frac{2}{9} \]

    第二步，求 \(E(X) = M_X'(0)\)。记 \(N(s) = 3+4e^{2s}+2e^{3s}\)，\(D(s) = 3-e^s\)，由商法则：

    \[ M_X'(s) = c\cdot\frac{N'(s)D(s) - N(s)D'(s)}{D(s)^2} = c\cdot\frac{(8e^{2s}+6e^{3s})(3-e^s) + (3+4e^{2s}+2e^{3s})e^s}{(3-e^s)^2} \]

    代入 \(s=0\)：分子 \(= (8+6)\times 2 + 9\times 1 = 28 + 9 = 37\)，分母 \(= 4\)，故

    \[ E(X) = \frac{2}{9}\cdot\frac{37}{4} = \frac{37}{18} \]

    第三步，求 \(p_X(1)\)。对 \(\lvert e^s\rvert < 3\) 用几何级数：

    \[ \frac{1}{3-e^s} = \frac{1}{3}\cdot\frac{1}{1-e^s/3} = \sum_{k=0}^{\infty} \frac{e^{ks}}{3^{k+1}} \]

    于是

    \[ M_X(s) = \frac{2}{9}\left[3\sum_{k=0}^{\infty}\frac{e^{ks}}{3^{k+1}} + 4\sum_{k=0}^{\infty}\frac{e^{(k+2)s}}{3^{k+1}} + 2\sum_{k=0}^{\infty}\frac{e^{(k+3)s}}{3^{k+1}}\right] \]

    由 \(M_X(s) = \sum_j p_X(j)e^{js}\)，取 \(e^{1\cdot s}\) 的系数：仅第一个和式 \(k=1\) 项贡献（后两个和式的最低次分别是 \(e^{2s}\) 与 \(e^{3s}\)）：

    \[ p_X(1) = \frac{2}{9}\cdot 3\cdot\frac{1}{3^2} = \frac{2}{27} \]

    第四步，求 \(E(X\mid X\ne 0)\)。先求 \(\mathbb{P}(X=0)\)：由性质 \(\mathbb{P}(X=0) = \lim_{s\to-\infty} M_X(s) = \dfrac{2}{9}\cdot\dfrac{3+0+0}{3-0} = \dfrac{2}{9}\)（也可读 \(e^{0s}\) 系数：\(\frac29\cdot 3\cdot\frac13 = \frac29\)，一致）。由于 \(X\ge 0\) 且 \(X\cdot\mathbf{1}_{\{X=0\}} = 0\)，全期望公式给出 \(E(X) = E(X\mid X\ne 0)\,\mathbb{P}(X\ne 0) + 0\)，故

    \[ E(X \mid X \ne 0) = \frac{E(X)}{\mathbb{P}(X\ne 0)} = \frac{37/18}{1 - 2/9} = \frac{37/18}{7/9} = \frac{37}{14} \]

    答：\(E(X) = \dfrac{37}{18}\)，\(p_X(1) = \dfrac{2}{27}\)，\(E(X\mid X\ne 0) = \dfrac{37}{14}\)。

    [!TIP] 技巧/易错 取非负整值的随机变量，其 MGF 就是把 PGF 中 \(s\) 换成 \(e^s\)，所以"\(e^{ks}\) 的系数 = \(\mathbb{P}(X=k)\)"。遇到分母 \(a - e^s\) 一律用几何级数展开；别忘了先用 \(M(0)=1\) 定出未知常数。

    题 5.7（来源：作业 HW11 第 6 题，教材 4.36）¶

    题目：设 \(X,Y,Z\) 相互独立，\(X \sim B(1,\tfrac13)\)，\(Y \sim \mathcal{E}(2)\)，\(Z \sim \mathcal{P}(3)\)。 (a) 令 \(U = XY + (1-X)Z\)，求 \(U\) 的矩母函数； (b) 求 \(2Z+3\) 的矩母函数； (c) 求 \(Y+Z\) 的矩母函数。

    [!EXAMPLE]- 详细解答 方法点拨：(a) 按 \(X\) 取值分情况用全期望公式（混合结构）；(b) 线性变换公式；(c) 独立和相乘。

    解：先写出零件：\(M_Y(s) = \dfrac{2}{2-s}\)（\(s<2\)），\(M_Z(s) = e^{3(e^s-1)}\)。

    (a) \(X\) 只取 \(0,1\)：\(X=1\)（概率 \(\tfrac13\)）时 \(U = Y\)；\(X=0\)（概率 \(\tfrac23\)）时 \(U = Z\)。由全期望公式并利用 \(X\) 与 \(Y,Z\) 独立：

    \[ \begin{aligned} M_U(s) &= E(e^{sU}) = E(e^{sU}\mid X=1)\,\mathbb{P}(X=1) + E(e^{sU}\mid X=0)\,\mathbb{P}(X=0) \\ &= \frac{1}{3}E(e^{sY}) + \frac{2}{3}E(e^{sZ}) = \frac{1}{3}\cdot\frac{2}{2-s} + \frac{2}{3}e^{3(e^s-1)}, \quad s<2 \end{aligned} \]

    即 \(U\) 是 \(\mathcal{E}(2)\) 与 \(\mathcal{P}(3)\) 的混合（权重 \(\tfrac13,\tfrac23\)）。

    (b) 由线性变换性质 \(M_{aX+b}(s) = e^{sb}M(as)\)，取 \(a=2,b=3\)：

    \[ M_{2Z+3}(s) = e^{3s}\,M_Z(2s) = e^{3s}\,e^{3(e^{2s}-1)} \]

    (c) \(Y\) 与 \(Z\) 独立，由独立和性质 MGF 相乘：

    \[ M_{Y+Z}(s) = M_Y(s)\,M_Z(s) = \frac{2}{2-s}\,e^{3(e^s-1)}, \quad s<2 \]

    答：\(M_U(s) = \dfrac13\cdot\dfrac{2}{2-s} + \dfrac23 e^{3(e^s-1)}\)；\(M_{2Z+3}(s) = e^{3s+3(e^{2s}-1)}\)；\(M_{Y+Z}(s) = \dfrac{2}{2-s}e^{3(e^s-1)}\)。

    [!TIP] 技巧/易错 (a) 中 \(U=XY+(1-X)Z\) 是随机选择开关结构，结果是混合分布的 MGF（凸组合），不要误用"独立乘积"把三个 MGF 直接相乘。(c) 的 \(Y+Z\) 一个连续一个离散，和的 MGF 照样相乘，但它不属于任何常见分布族，写出乘积即可。

    题 5.8（来源：作业 HW11 第 2 题 + 样卷 题六）¶

    题目：设 \(X_1,X_2,\ldots\) 是独立同分布序列，\(X_i \sim B(1,p)\)，\(N\) 取正整数值且与 \(\{X_i\}\) 独立。记 \(T = X_1 + \cdots + X_N\)。 (1) 若 \(N \sim \mathcal{P}(\lambda)\)，求 \(T\) 服从什么分布（写出参数与推导）； (2) 若 \(N \sim G(\tilde p)\)（几何分布），求 \(E(T)\)。

    [!EXAMPLE]- 详细解答 方法点拨：随机和定理 \(g_T(s) = g_N[g_X(s)]\)，复合后比对总表；求期望可对复合式在 \(s=1\) 求导（即 Wald 公式 \(E(T)=E(N)E(X)\)）。

    解：(1) 第一步，写出两个 PGF：\(g_X(s) = q + ps\)（\(q=1-p\)），\(g_N(s) = e^{\lambda(s-1)}\)。

    第二步，由随机和定理（\(N\) 与 \(\{X_i\}\) 独立，\(X_i\) 独立同分布），

    \[ g_T(s) = g_N\bigl[g_X(s)\bigr] = e^{\lambda(q+ps-1)} \]

    第三步，化简指数：\(q + ps - 1 = (1-p) + ps - 1 = p(s-1)\)，故

    \[ g_T(s) = e^{\lambda p(s-1)} \]

    第四步，这正是 \(\mathcal{P}(\lambda p)\) 的 PGF，由 PGF 可逆性，\(T \sim \mathcal{P}(\lambda p)\)。

    (2) 第一步，对复合式求导（链式法则）：

    \[ g_T'(s) = g_N'\bigl[g_X(s)\bigr]\cdot g_X'(s) \]

    第二步，代 \(s=1\) 并用 \(g_X(1)=1\)：

    \[ E(T) = g_T'(1) = g_N'(1)\,g_X'(1) = E(N)\,E(X_1) \]

    （这就是 Wald 公式。）

    第三步，代入数值：\(E(N) = \dfrac{1}{\tilde p}\)（几何分布期望），\(E(X_1) = p\)（Bernoulli 期望），故

    \[ E(T) = \frac{p}{\tilde p} \]

    答：(1) \(T \sim \mathcal{P}(\lambda p)\)（Poisson 的"稀疏化"）；(2) \(E(T) = \dfrac{p}{\tilde p}\)。

    [!TIP] 技巧/易错 复合方向别搞反：是把 \(g_X(s)\) 代入 \(g_N\) 的变量位置（外层是 \(N\)），即 \(g_N[g_X(s)]\) 而非 \(g_X[g_N(s)]\)。记法：先"打包"一个 \(X\)，再按 \(N\) 的结构复制。同类经典结果：\(X_i\sim G(p)\)、\(N\sim G(\tilde p)\) 时 \(T \sim G(p\tilde p)\)；\(X_i\sim\mathcal{E}(\lambda)\)、\(N\sim G(p)\) 时（MGF 版）\(T \sim \mathcal{E}(p\lambda)\)。

    题 5.9（来源：作业 HW12 第 2 题）¶

    题目：设 \(\{Y_k,\,k=1,2,\ldots\}\) 独立同分布且方差有限，用 \(\phi_Y(t) = E(e^{itY_1})\) 表示 \(Y_1\) 的特征函数。请基于 \(\phi_Y(t)\) 写出

    \[ Z_n = \frac{Y_1 + \cdots + Y_n}{\sqrt{n}} \]

    的特征函数 \(\phi_{Z_n}(t)\)。

    [!EXAMPLE]- 详细解答 方法点拨：独立和的 CF 相乘 + 线性变换 \(\phi_{aX}(t) = \phi_X(at)\)。

    解：第一步，记 \(S_n = Y_1+\cdots+Y_n\)。由 \(Y_1,\ldots,Y_n\) 相互独立及独立和性质（性质 5）：

    \[ \phi_{S_n}(t) = \prod_{k=1}^{n}\phi_{Y_k}(t) = \bigl[\phi_Y(t)\bigr]^n \]

    （同分布故各因子相同。）

    第二步，\(Z_n = a S_n\)，\(a = \dfrac{1}{\sqrt n}\)。由线性变换性质（性质 4，\(b=0\)）：

    \[ \phi_{Z_n}(t) = \phi_{S_n}\!\left(\frac{t}{\sqrt n}\right) \]

    第三步，合并：

    \[ \phi_{Z_n}(t) = \left[\phi_Y\!\left(\frac{t}{\sqrt{n}}\right)\right]^{n} \]

    答：\(\phi_{Z_n}(t) = \bigl[\phi_Y(t/\sqrt{n})\bigr]^n\)。

    [!TIP] 技巧/易错 这是 CLT 证明的起点：若再设 \(E(Y_1)=0\)、\(\operatorname{var}(Y_1)=1\)，用矩展开 \(\phi_Y(u) = 1 - \tfrac{u^2}{2} + o(u^2)\) 代入得 \(\phi_{Z_n}(t) = \bigl(1-\tfrac{t^2}{2n}+o(\tfrac1n)\bigr)^n \to e^{-t^2/2}\)，由连续性定理 \(Z_n \xrightarrow{d} \mathcal{N}(0,1)\)。注意线性变换是把 \(a\) 乘进自变量：\(\phi_{aX}(t)=\phi_X(at)\)，不是 \(a\,\phi_X(t)\)。

    题 5.10（补充题）¶

    题目：设 \(X_1,\ldots,X_m\) 相互独立，\(X_j \sim \mathcal{N}(\mu_j,\sigma_j^2)\)。用特征函数（或矩母函数）证明正态分布的再生性：

    \[ Y = X_1 + \cdots + X_m \sim \mathcal{N}\!\left(\sum_{j=1}^m \mu_j,\ \sum_{j=1}^m \sigma_j^2\right) \]

    [!EXAMPLE]- 详细解答 方法点拨：独立和的 CF 相乘，指数相加后比对正态 CF，再引用唯一性。证明题模板：写零件、乘起来、认形状、引可逆性，四步缺一不可。

    解：第一步，写出每个 \(X_j\) 的特征函数（正态 CF，可直接引用）：

    \[ \phi_{X_j}(t) = \exp\!\left(i\mu_j t - \frac{1}{2}\sigma_j^2 t^2\right) \]

    第二步，由 \(X_1,\ldots,X_m\) 相互独立及独立和性质（性质 5）：

    \[ \phi_Y(t) = \prod_{j=1}^{m}\phi_{X_j}(t) = \exp\!\left(\sum_{j=1}^m \left(i\mu_j t - \frac{1}{2}\sigma_j^2 t^2\right)\right) = \exp\!\left(i\Bigl(\sum_{j=1}^m\mu_j\Bigr)t - \frac{1}{2}\Bigl(\sum_{j=1}^m\sigma_j^2\Bigr)t^2\right) \]

    第三步，上式恰为 \(\mathcal{N}\bigl(\sum_j \mu_j,\ \sum_j \sigma_j^2\bigr)\) 的特征函数。

    第四步，由特征函数与分布函数相互唯一决定（唯一性定理），

    \[ Y \sim \mathcal{N}\!\left(\sum_{j=1}^m \mu_j,\ \sum_{j=1}^m \sigma_j^2\right) \qquad \blacksquare \]

    （若用 MGF：\(M_Y(s) = \prod_j e^{\mu_j s + \sigma_j^2 s^2/2} = e^{(\sum\mu_j)s + (\sum\sigma_j^2)s^2/2}\)，且在 \(0\) 的邻域内有限，由 MGF 可逆性同样得证。）

    答：\(Y \sim \mathcal{N}\bigl(\sum_j \mu_j,\ \sum_j \sigma_j^2\bigr)\)，证毕。

    [!TIP] 技巧/易错 写证明时"由唯一性/可逆性"这一步必须显式写出，否则只证明了"CF 相同"，没证明"分布相同"。同样的四步模板可证：独立 Poisson 之和是 Poisson（参数相加）、独立同 \(p\) 二项之和是二项（次数相加）、独立 Gamma（同 \(\lambda\)）之和是 Gamma（形状参数相加）。

    Part D：模块小结¶

    一条主线：母函数把"分布的卷积"变成"函数的乘法"，把"求矩"变成"求导/读系数"。三种工具按适用范围递增：PGF（非负整值）\(\subset\) MGF（需存在）\(\subset\) CF（恒存在），三者通过替换 \(s \to e^s \to e^{it}\) 互通。

    综合套路：全期望公式 + PGF 求随机和分布（本模块与模块 4 条件期望的交汇，样卷题六即此型）：

    \[ g_W(s) = E(s^W) = E\bigl[E(s^W \mid N)\bigr] = \sum_{n\ge1}\bigl[g_X(s)\bigr]^n\,\mathbb{P}(N=n) = g_N\bigl[g_X(s)\bigr] \]

    配套结论：两边求导得 \(E(W) = E(N)E(X)\)（Wald 公式），考试中若只问 \(E(W)\) 可直接用它，但要先声明 \(N\) 与 \(\{X_i\}\) 独立、\(X_i\) 独立同分布。

    与模块 6 的衔接：连续性定理（\(X_n \xrightarrow{d} X \iff \phi_n(t)\to\phi(t)\) 逐点）是 CLT 证明的桥梁，题 5.9 的 \(\bigl[\phi_Y(t/\sqrt n)\bigr]^n\) 正是其标准入口；样卷题七"\(T_n/n\) 的 MGF 极限"同理：\(M_n(t) = \bigl[M_X(t/n)\bigr]^n \to e^{tE(X)}\)，即退化分布（大数定律的母函数版本）。

    考前自查清单：① 三个归一点 \(g(1)=M(0)=\phi(0)=1\)；② 求矩位置 PGF 在 \(1\)、MGF 在 \(0\)；③ 可逆性引用前提（MGF 需邻域内有限）；④ 随机和复合方向 \(g_N[\,\cdot\,]\)；⑤ Cauchy 反例（MGF 不存在；\(\phi_{X+Y}=\phi_X\phi_Y\) 不推独立）。
